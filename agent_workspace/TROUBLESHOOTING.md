# Troubleshooting

## find_api ModuleNotFoundError

Symptom:
- `ModuleNotFoundError: No module named 'find_api'`
- The failure appears while running `03_figure_mapping`, `03_split_excel_blocks_batch`, `04_figure_separate`, or `04_ft_excel_matcher`.

Cause:
- A legacy Gemini/API script was imported.
- This should not happen in API-free `external_agent` or `heuristic` mode.

Fix:
- Check `STAGE_EXECUTION_MODE` in `Agent_Task_Runner.py`.
- Set the stage mode to `external_agent` or `heuristic`.
- Do not run `legacy` mode unless `find_api.py`, old Gemini helper modules, and credentials are available.

## Active Stage Blocked by Missing Manual Marker

Symptom:
- Runner or agent refuses `03_*` or `04_*` stages.

Check:
- `<PAPER_FOLDER>/.manual_select_review_done`
- `fig_table_lnpdb_classified.csv`
- `fig_table_lnpdb_classified_manual_reviewed.csv`

Action:
- Run the Streamlit review app and have a human save review results.
- Do not bypass this gate unless explicitly instructed by the project owner.

## Legacy Script Uses Hard-Coded Path

Symptom:
- Script runs but processes an unrelated folder.
- Output appears under an old `/Users/kogeon/...` or `C:\Users\kogun\...Extraction_Examples...` path.

Action:
- Prefer `Agent_Task_Runner.py`.
- If direct execution is necessary, modify only the execution block or call the script functions via import.
- Log the exact path used.

## `04_figure_separate_gemini.py` Not Found

Symptom:
- Agent tries to run the filename from older notes and gets file-not-found.

Cause:
- Current repository filename is `0_mark_down_gen/04_figure_saperate_gemini.py`.

Action:
- Use the actual filename.
- Keep the stage name `04_figure_separate` in state files for readability.

## API Credentials Missing

Symptom:
- `vertex.json`, `gemini_api.txt`, or service account errors.
- Project ID missing.

Action:
- Do not hard-code secrets.
- Check whether credential files exist where `find_api.py` expects them.
- If credentials are unavailable, mark the task blocked with the missing credential name.

## Batch Job Repeated or Expensive

Symptom:
- Stage creates duplicate `_batch_jobs` entries or resubmits cloud work.

Action:
- Inspect existing `_batch_jobs/` files before retry.
- Validate whether prior output can be downloaded/reused.
- Log the batch job ID and retry reason.

## Empty Figure/Table Classification

Symptom:
- `fig_table_lnpdb_classified.csv` exists but no selected rows are available.

Action:
- Check inventory quality and captions.
- Request human review if source figures/tables look relevant.
- Do not continue to mapping stages with a clearly wrong empty selection.

## Excel Block Paths Missing

Symptom:
- `excel_block_inventory.csv` exists but listed block CSVs do not.

Action:
- Resolve `block_csv_path` relative to `<PAPER_FOLDER>`.
- Check `Exp_Excel_Blocks/`.
- Rerun `03_split_excel_blocks_batch` only after checking whether save failed or inventory paths are stale.

## Mapping JSON Empty

Symptom:
- `total_figure_mapping.json` parses but has no useful entries.

Action:
- Confirm selected FT rows after manual review.
- Check source folders and images generated by Marker.
- Confirm `03_figure_mapping.py` scanned the intended root.
