139:In this study, we engineered a synergistic immunotherapy platform, G0-SS-AA-C12 IL-12 pLNP, which showed robust anti-tumour efficacy. Further optimization of this therapeutic platform could be accomplished by balancing pLNP stability and IDO inhibitor release. Several factors influence pLNP stability, including the tail length of pIL and the molar ratio of the helper lipids in the LN[P50,](#page-11-28) which modulate the intramolecular forces that drive pLNP formulation. The chemical structures of constituent lipids within pLNPs are likely to influence IDO release kinetics. Different linker chemistries may be designed to respond to specific biological triggers in the TME, such as pH, redox markers (for example, reactive oxygen species (ROS) and GSH) or enzymatic activity[41](#page-11-20). While in the present study we utilized a cyclic reaction for IDO inhibitor release, self-immolative linkers based on 1,6- or 1,4-azaquinone methide elimination could also be used to precisely control IDO inhibitor release kinetics[51](#page-11-29). Optimization of these parameters may identify a next-generation pLNP formulation that balances stability and IDO inhibitor release to achieve superior therapeutic outcomes.
141:Intravenous administration of G0-SS-AA-C12 IL-12 pLNPs demonstrated significant anti-tumour efficacy (Supplementary Fig. 86) but was accompanied by systemic toxicity, including cytokine release syndrome and hepatotoxicity in mice (Supplementary Figs. 87 and 88). To mitigate these adverse effects, several strategies could be implemented. For example, incorporating microRNA-122 target sequences (miR-122T) into the 3′-untranslated region of IL-12 mRNA may substantially reduce hepatic IL-12 expressio[n52](#page-11-30). Alternatively, the use of pro-IL-12 mRNA could generate an inactive precursor that is specifically activated within the TME. Conjugating a tumour-targeting antibody to pLNPs may further enhance tumour localization while minimizing hepatic transfection. Finally, co-administration of inhibitors targeting IL-6, TNF-α or IFNγ could reduce IL-12-induced toxicity without compromising anti-tumour efficacy.
145:A critical requirement for the success of intratumoural immunotherapy is treating metastatic tumours via generation of a systemic anti-tumour immune response, referred to as the abscopal effect. This requires the intratumoural agent to trigger events leading to 'in situ' vaccination[58,](#page-12-3) that is, the generation or expansion of tumour-specific T cells that not only attack the tumour at the site of injection but also circulate and target distant, non-injected tumours. Encouragingly, in the present study, we observed that intratumoural injection of G0-SS-AA-C12 pLNP encapsulating IL-12 mRNA resulted in systemic (splenic) T cell activation (Fig. [4](#page-5-0)), immunologic memory (Figs. [4i–l](#page-5-0) and [6a–e\)](#page-8-0) and, importantly, a strong abscopal effect (Fig. [6f–j](#page-8-0)). These data support the use of pLNPs for disseminated cancers after local intratumoural injection.

def canonical_smiles(smiles: str) -> tuple[str, str]:
    if not smiles:
        return "", ""
    if Chem is None:
        return smiles, "rdkit_unavailable_not_canonicalized"
    mol = Chem.MolFromSmiles(smiles)
    if mol is None:
        return "", "invalid_curated_smiles"
    return Chem.MolToSmiles(mol, isomericSmiles=True), ""


def read_sources() -> list[dict[str, str]]:
    rows: list[dict[str, str]] = []
    for path in TEXT_SOURCES:
        if not path.exists():
            continue
        rel = path.relative_to(PAPER_FOLDER).as_posix()
        for line_no, line in enumerate(path.read_text(encoding="utf-8", errors="replace").splitlines(), 1):
            text = re.sub(r"\s+", " ", line).strip()
            if text:
                rows.append({"source_path": rel, "line": str(line_no), "text": text})
    return rows


def find_evidence(name: str, sources: list[dict[str, str]]) -> dict[str, str]:
    variants = {name, name.replace("G0-", "GO-"), name.replace("GO-", "G0-")}
    if name == "DOPE":
        variants.add("1,2-dioleoyl")
    if name == "DMG-PEG 2000":
        variants.add("DMG-PEG")
    if name == "DLin-MC3-DMA":
        variants.add("MC3")
    if name == "indoximod":
        variants.add("1-methyl-D-tryptophan")
    for row in sources:
        hay = row["text"]
        if any(v and v in hay for v in variants):
            return {
                "source_path": row["source_path"],
                "source_line": row["line"],
                "evidence_text": hay[:700],
            }
    return {"source_path": "", "source_line": "", "evidence_text": ""}
